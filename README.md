## 开发说明：
### 一、本demo使用技术栈
    1.H5：支持移动端、PC端
    2.css：使用css3高级动画，animate.css框架动画
    3.Javascript：ES5语法
    4.jQuery: 选择器、循环API、轮播
   
### 二、本demo各功能
    1.支持移动端、pc端
    2.支持一次分享、二次分享
    3.支持外部分享页进入后展示下载横幅
    4.支持PV人数展示
    5.如若想编辑本demo代码，拉取code，在dev分支下提交您的code，我会合并至master
